(4) Watkins Exercise 3.2.12

QTx = [y, 0]

(4)-(a)

Give the geometric interpretation of this fact.

This satisfies if and only if x * sin(theta) = y * cos(theta). Therefore, tan(theta) = x/y. This always holds true. Geometrically, QTx = [y, 0] means that QT matrix is an linear transformation where it transforms any coordinate x=(x1, x2) in to (sqrt(xTx), 0), which is a point on the y-axis intersecting the circle x is in.

(4)-(b)

Show that cos(theta)=x1/sqrt(x12+x22), sin(theta)=x2/sqrt(x12+x22), then y = ||x||2

y = cos(theta)x1+sin(theta)x2 When we substitute cos and sin, then we get y = x12/sqrt(x12+x2^2) + x22/sqrt(x12+x2^2) = sqrt(x12+x22) = ||x||2

Implementation of Gram-Schmidt

# classical Gram-Schmidt algorithm

euc_norm <- function(x) sqrt(sum(x^2))

classic_GS <- function(A){
  r = matrix(nrow=dim(A)[1], ncol=dim(A)[2])
  for (k in 1:dim(A)[2]){
    if(euc_norm(A[,k])==0){
      print('Length of column is 0. End of function.')
      stop()
    } # break out of loop if length is 0
    k_vec=1:k
    for (i in k_vec[-k]){
      r[i,k] <- sum(A[,i]*A[,k])
      A[,k] <- A[,k] - r[i,k]*A[,i]
    }
    A[,k] <- A[,k]/(euc_norm(A[,k])) 
  } 
  return(A)
}
